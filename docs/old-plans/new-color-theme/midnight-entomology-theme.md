# Midnight Entomology — Custom Theme

A customized theme blending the depth of Midnight Galaxy with the scientific precision of entomological illustration. Museum-cabinet aesthetic with rich, dark navigation and warm ochre accents.

## Color Palette

### Navigation (Dark Cabinet)
- **Primary Dark**: `#1a1714` - Deep umber black
- **Secondary Dark**: `#2c2418` - Rich iron brown
- **Tertiary Dark**: `#3a3025` - Warm charcoal
- **Text Primary**: `#d4c9b0` - Warm parchment
- **Text Secondary**: `#8a7e6b` - Muted marginalia
- **Text Active**: `#f5f0e1` - Bright parchment

### Content Area (Light Paper)
- **Parchment**: `#f5f0e1` - Warm paper base
- **Parchment Dark**: `#ebe5d4` - Aged paper

### Accent Colors
- **Ochre**: `#c4963a` - Golden highlight (primary accent)
- **Ochre Light**: `#d4aa55` - Bright gold
- **Ochre Glow**: `rgba(196, 150, 58, 0.4)` - Subtle radiance
- **Verdigris**: `#4a8c7e` - Copper patina (secondary accent)
- **Carmine**: `#9b2335` - Insect red (tertiary accent)

### Borders & Rules
- **Rule**: `#d4c9b0` - Light divider
- **Rule Dark**: `#4a4035` - Dark divider
- **Rule Accent**: `rgba(196, 150, 58, 0.3)` - Golden edge

### Shadows
- **Shadow**: `rgba(26, 23, 20, 0.08)` - Whisper
- **Shadow Medium**: `rgba(26, 23, 20, 0.24)` - Present
- **Shadow Heavy**: `rgba(26, 23, 20, 0.48)` - Deep
- **Shadow Glow**: `rgba(196, 150, 58, 0.16)` - Warm radiance

## Typography

- **Display/Headings**: Playfair Display — Elegant, editorial serif
- **Body Text**: EB Garamond — Classic, readable serif
- **Accent/Quotes**: Cormorant Garamond — Refined italic serif
- **Numbers/Labels**: Source Code Pro — Precise monospace

## Design Principles

1. **Dark Navigation, Light Content**: Clear visual separation
2. **Ochre Accents**: Golden highlights draw the eye to interactive elements
3. **Museum-Quality**: Every element feels intentional and premium
4. **Scientific Precision**: Clear hierarchy, readable at all sizes
5. **Atmospheric Depth**: Subtle gradients and shadows create layers

## Best Used For

Scientific publications, natural history presentations, museum exhibits, academic journals, art books.
